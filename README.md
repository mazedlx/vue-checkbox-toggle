# vue-checkbox-toggle

A simple VueJS checkbox replacment thing.

## Installation

First, install it with NPM

`npm install vue-checkbox-toggle`

Then, import it where needed

`import vue-checkbox-toggle from 'vue-checkbox-toggle';`

Finally, use it like

```
<vue-checkbox-toggle
    v-model="val"
    showLabels
    labelCheck="on"
    labelUnchecked="off"
></vue-checkbox-toggle>```
